# Brand Packaging Portfolio

一个给品牌包装设计师使用的个人作品集网站，视觉方向参考 Clay 的暖纸感与彩色手作气质，并用 Apple 式克制留白、产品叙事和清晰层级来组织内容。

## Tech Stack

- Vite
- React
- TypeScript
- lucide-react

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Content

- Hero packaging mockups
- Selected packaging case studies
- Studio method and services
- Contact CTA

图片资产位于 `public/projects/`，核心页面在 `src/App.tsx`，视觉系统在 `src/App.css` 与 `src/index.css`。
